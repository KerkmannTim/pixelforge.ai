# PixelForge.ai

KI-Stockfoto-Plattform mit dynamischem Preismodell.

## Deploy auf Vercel

1. https://vercel.com/new
2. Import Git Repository
3. pixelforge.ai auswählen
4. Deploy

## Environment Variables (in Vercel Settings)

Siehe .env.local.example für die benötigten Variablen.

## Google OAuth

Siehe Supabase Dashboard → Auth → Providers → Google.
